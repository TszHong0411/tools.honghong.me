## 🌍 tools.honghong.me

- 框架: [Next.js](https://nextjs.org/)
- 部署: [Vercel](https://vercel.com)
- 網站圖標: [realfavicongenerator](https://realfavicongenerator.net/)

## 🤩 特色

- CSS 單位轉換器
- 顏色轉換器
- 圖片格式轉換器
- PDF 閱讀器
- 字數計算器
- 鍵盤測試器

## 👋 在本地運行

1. Clone 這個倉庫

```sh
git clone https://github.com/tszhong0411/tools.git
```

2. 更改目錄

```sh
cd tools.honghong.me
```

3. 安裝 packages

```sh
yarn
```
